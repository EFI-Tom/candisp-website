# CanDisp Website

Official product website for the CanDisp CAN Bus Display.

## About CanDisp

CanDisp is a compact standalone CAN Bus display for real-time vehicle data visualization.

Features include:

* Real-time CAN Bus data display
* 1.85" round IPS display
* Configurable gauge layouts
* Wi-Fi based configuration interface
* Passive CAN Bus listener operation
* Google Maps turn-by-turn integration via BLE

The system is designed for development, tuning and motorsport applications.
